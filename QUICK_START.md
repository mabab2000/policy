## üéØ First-Time User Guide

### For Ministry Officers

1. **Create a Policy**
   - Click "Policies" in sidebar
   - Click "Create Policy" button
   - Fill in all required fields:
     - Policy title and code
     - Description (minimum 50 characters)
     - Problem statement
     - Objectives (one per line)
     - Vision 2050 alignment
     - NST alignment
     - Select your ministry
     - Set priority level
   - Click "Create Policy"

2. **View Dashboard**
   - Click "Dashboard" in sidebar
   - See overview of all policies
   - View charts and statistics
   - Monitor performance metrics

3. **Manage Stakeholders**
   - Click "Stakeholders" in sidebar
   - Add stakeholder organizations
   - Track consultation feedback

### For District Officers

1. **View Assigned Activities**
   - Click "Execution" in sidebar
   - See activities assigned to your district
   - Track progress on each activity

2. **Submit Progress Reports**
   - Click on an activity
   - Click "Submit Progress Report"
   - Fill in achievements, challenges, and next steps
   - Upload evidence if available

3. **Monitor Imihigo**
   - Click "Imihigo" in sidebar
   - View your district's performance contract
   - Track KPIs and targets

### For Citizens

1. **Browse Policies**
   - Click "Policies" in sidebar
   - Search and filter policies
   - Click on a policy to view details

2. **Submit Feedback**
   - Click "Citizen Feedback" in sidebar
   - Click "Submit Feedback" button
   - Choose feedback type:
     - General Feedback
     - Suggestion
     - Complaint
     - Appreciation
   - Write your message
   - Submit

3. **Track Your Feedback**
   - View all your submitted feedback
   - See response status
   - Read government responses

## üé® Using the Interface

### Navigation
- **Sidebar**: Main navigation menu (changes based on role)
- **Header**: User profile and logout
- **Breadcrumbs**: Current location in the app

### Colors Meaning
- **Blue**: Primary actions and information
- **Green**: Success, completion, positive trends
- **Yellow**: Warnings, pending actions
- **Red**: Errors, critical issues

### Buttons
- **Primary (Blue)**: Main actions
- **Success (Green)**: Confirm actions
- **Warning (Yellow)**: Caution actions
- **Outline**: Secondary actions
- **Ghost**: Tertiary actions

### Filters
Most list pages have filters:
1. Search by text
2. Filter by status
3. Filter by priority
4. Sort by date or name

## üÜò Troubleshooting

### Login Issues
- Check email format
- Verify password (all demo accounts use: password123)
- Clear browser cache if needed

### Page Not Loading
- Refresh the page
- Check internet connection
- Verify you have access to that page (role-based)

### Data Not Showing
- Check filters (might be too restrictive)
- Verify date range
- Try resetting filters

### Form Errors
- Check all required fields (marked with *)
- Ensure minimum character requirements
- Verify email format if applicable
